# C++ Pattern Problems

This repository contains my solutions to the **Pattern Problems** from **Striver's A2Z DSA Course**. These exercises are my first step before starting Data Structures & Algorithms and are aimed at strengthening my understanding of nested loops, pattern recognition, and problem-solving in C++.

## Source

The problems are taken from **Striver's A2Z DSA Course**:

https://takeuforward.org/strivers-a2z-dsa-course/must-do-pattern-problems-before-starting-dsa

---

## Objectives

While solving these problems, I focused on:

- Understanding nested loops
- Improving pattern observation skills
- Finding mathematical relationships instead of hardcoding logic
- Writing clean and readable C++ code
- Reducing duplicate code where appropriate
- Practicing modular programming using functions

---

## Concepts Practiced

- Nested loops
- Conditional statements
- Character arithmetic
- Mathematical pattern derivation
- Coordinate-based thinking
- Symmetry in patterns
- Function decomposition
- Code refactoring
- Time and Space Complexity analysis
